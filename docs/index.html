Perplexity WebUI Search

A portable Agent Skill that queries Perplexity through its web UI using Playwright — no API key required.

Get Started View on GitHub
Node.js Runtime
Playwright Framework
MIT License

Features

Browser Automation

Launches Chromium, navigates Perplexity, submits queries — no API key needed.

Answer Extraction

Clicks "Copy" on the generated response and reads from clipboard with retry logic.

Markdown Output

Saves answers as .md files with clean stdout/stderr separation.

Multi-Agent Ready

Works with OpenCode, Claude Code, and Codex CLI out of the box.

Tested

Unit test suite covering arg parsing, retry logic, and copy-with-retries behavior.

Quick Start

Get up and running in under 60 seconds.
